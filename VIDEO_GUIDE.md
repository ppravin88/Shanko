# Video Recording Guide for Shanko Card Game

This comprehensive guide provides instructions for recording high-quality gameplay and promotional videos for Shanko.

## Video Types Overview

1. **Quick Promo Video** (30-60 seconds) - Social media and website hero
2. **Tutorial Video** (3-5 minutes) - Teaching new players
3. **Strategy Guide** (5-10 minutes) - Advanced tips and tactics
4. **Gameplay Showcase** (2-3 minutes) - Full round demonstration
5. **Feature Highlights** (15-30 seconds each) - Individual mechanics

## Equipment and Software

### Required

- **Computer** with game running smoothly
- **Screen recording software** (see options below)
- **Modern web browser** (Chrome or Firefox recommended)
- **Stable internet connection** (for initial game load)

### Optional but Recommended

- **Microphone** for voiceover (Blue Yeti, Audio-Technica AT2020)
- **Video editing software** for post-production
- **Background music** (royalty-free from YouTube Audio Library)
- **Script** for narration (see scripts below)
- **Teleprompter software** (if recording voiceover)

## Video 2: Tutorial Video (3-5 minutes)

### Purpose
Teach new players how to play Shanko from start to finish.

### Target Audience
- Complete beginners
- Players familiar with Rummy who want to learn Shanko's unique mechanics
- Visual learners

### Script

**[0:00-0:15] Introduction**

*Voiceover:*
"Welcome to Shanko, a strategic card game that combines classic Rummy with unique mechanics. In this tutorial, you'll learn everything you need to start playing. Let's dive in!"

*Visual:*
- Game logo
- Transition to game setup screen

**[0:15-0:45] Game Setup**

*Voiceover:*
"First, choose your player count. Shanko supports 2 to 8 players. You can play with friends or against AI opponents. For this tutorial, we'll set up a 4-player game with 2 humans and 2 AI players."

*Visual:*
- Show player count selector
- Configure players
- Show deck count indicator
- Click "Start Game"

**[0:45-1:30] Basic Rules**

*Voiceover:*
"The goal is simple: form the required combinations for each round and achieve the lowest score across 7 rounds. There are two types of combinations:

Triplets: Three cards of the same rank, like three 7s.
Sequences: Four consecutive cards of the same suit, like 5, 6, 7, 8 of hearts.

Each round has different requirements. Round 1 requires 2 triplets. Round 2 requires 1 triplet and 1 sequence. And so on, getting progressively harder."

*Visual:*
- Show examples of valid triplets
- Show examples of valid sequences
- Display round objectives for all 7 rounds
- Highlight current round objective

**[1:30-2:15] Turn Structure**

*Voiceover:*
"On your turn, you'll do three things:

First, draw a card from either the draw pile or the discard pile.
Second, if you have the required combinations, you can meld them by clicking the Meld button.
Third, discard one card to end your turn.

Let's see this in action."

*Visual:*
- Show drawing from draw pile
- Show drawing from discard pile
- Show hand after drawing
- Show meld button
- Show discarding a card

**[2:15-3:00] The Buying Mechanic**

*Voiceover:*
"Here's where Shanko gets interesting. When another player discards a card you need, you can 'buy' it even when it's not your turn!

To buy, click the Buy button. You'll receive the discarded card plus one card from the draw pile. But be strategic‚Äîyou only get 3 buys per round!

Note: Buying is disabled in 2-player games, and once you meld, you can't buy anymore."

*Visual:*
- Show another player discarding
- Show Buy button appearing
- Show buy dialog
- Show receiving both cards
- Show buys remaining counter

**[3:00-3:30] Jokers**

*Voiceover:*
"Jokers are wild cards that can substitute for any card in your combinations. They're powerful but risky‚Äîif you're caught with one in your hand, it's worth 50 points!

After melding, you can swap Jokers from sequences by replacing them with the actual card. This lets you reuse the Joker to extend other sequences."

*Visual:*
- Show Joker in a combination
- Show Joker swap dialog
- Show swapping process
- Show extending a sequence with the swapped Joker

**[3:30-4:00] Going Out and Scoring**

*Voiceover:*
"When you have all required combinations, you can 'go out' by melding and discarding your final card face-down. This ends the round, and you score zero points!

Other players score points based on cards left in their hand:
- Number cards: face value
- Face cards: 10 points
- Aces: 15 points
- Jokers: 50 points

After 7 rounds, the player with the lowest total score wins!"

*Visual:*
- Show going out process
- Show round end screen
- Show score calculation
- Show cumulative scores

**[4:00-4:30] Strategy Tips**

*Voiceover:*
"Here are some quick tips:

Save your buys for cards that complete combinations.
Use Jokers in sequences rather than triplets so you can swap them later.
Decide whether to meld early for safety or wait to surprise opponents.
Pay attention to what others are discarding to avoid helping them.

Now you're ready to play! Good luck!"

*Visual:*
- Show strategic buying
- Show Joker placement
- Show meld timing decision
- Show observing opponents

**[4:30-4:45] Closing**

*Voiceover:*
"Thanks for watching! Click the link below to start playing Shanko now. Have fun!"

*Visual:*
- Game logo
- Website URL
- "Play Now" button
- Subscribe/like prompts

### Recording Instructions

1. **Script Preparation**
   - Print script or use teleprompter
   - Practice voiceover multiple times
   - Time each section

2. **Game Scenarios**
   - Pre-play scenarios to capture
   - Have specific hands ready
   - Know what you'll demonstrate

3. **Recording Process**
   - Record voiceover separately (better quality)
   - Record gameplay footage
   - Sync in post-production

4. **Editing Checklist**
   - Cut out dead time
   - Add text overlays for key points
   - Add arrows/highlights to UI elements
   - Sync voiceover with visuals
   - Add background music (low volume)
   - Add captions/subtitles

### Technical Specs

- **Resolution**: 1920x1080 (1080p)
- **Frame Rate**: 30fps (sufficient for tutorial)
- **Format**: MP4 (H.264)
- **Length**: 3-5 minutes
- **Audio**: Clear voiceover + soft background music
- **Captions**: Full subtitles (accessibility)
- **File Size**: <100MB

## OBS Studio Setup Guide

### Installation

1. Download from https://obsproject.com
2. Install with default settings
3. Launch OBS Studio

### Configuration

**1. Settings ‚Üí Video**
- Base Resolution: 1920x1080
- Output Resolution: 1920x1080
- FPS: 60 (for promo) or 30 (for tutorial)

**2. Settings ‚Üí Output**
- Output Mode: Simple
- Video Bitrate: 6000 Kbps
- Encoder: x264
- Audio Bitrate: 160 Kbps
- Recording Format: MP4

**3. Settings ‚Üí Audio**
- Sample Rate: 48kHz
- Channels: Stereo
- Desktop Audio: Default
- Mic/Auxiliary Audio: Your microphone

### Scene Setup

**Scene 1: Full Screen Gameplay**
- Source: Browser (game URL)
- Size: 1920x1080
- Position: 0, 0

**Scene 2: Gameplay with Webcam**
- Source 1: Browser (game)
- Source 2: Video Capture Device (webcam)
- Position webcam in corner

**Scene 3: Logo/Intro**
- Source: Image (logo)
- Background color or image

### Recording Process

1. Select scene
2. Click "Start Recording"
3. Play through scenario
4. Click "Stop Recording"
5. Find video in default folder

## Troubleshooting

### Problem: Choppy Recording

**Solutions:**
- Lower recording quality
- Close other applications
- Update graphics drivers
- Use hardware encoding

### Problem: Audio Out of Sync

**Solutions:**
- Record audio separately
- Use constant frame rate
- Sync in post-production
- Check audio buffer settings

### Problem: Large File Sizes

**Solutions:**
- Lower bitrate
- Use more efficient codec (H.265)
- Compress after recording
- Trim unnecessary footage

### Problem: Poor Audio Quality

**Solutions:**
- Use external microphone
- Record in quiet room
- Use noise reduction software
- Adjust microphone position
